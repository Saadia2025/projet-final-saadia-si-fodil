Scénario de projet 

Vous avez été engagé en tant qu’ingénieur de données par une organisation de recherche. Vous devez télécharger le fichier CSV et appliquer des méthodes ETL en utilisant Python. À la fin, affichez les résultats après l’ETL dans un tableau. De plus, incluez une requête pour afficher les pays les plus riches, par exemple ceux dont le PIB est supérieur ou égal à 10 000 milliards de dollars. Le jeu de données final traité doit être stocké et affiché sous forme de table de base de données sur le serveur web de l’API.

Instructions:

  1. ETL avec Python - crée api.py

    • Utilise Pandas pour crée DataFrame
    • Utilise sqlite3 pour connecter la base de données SQLite
  1. Crée app.js pour un serveur Web API avec Express
    • Servir des fichiers statiques(HTML, CSS, JS)
    • Creation des routes
    • Démarrer le Serveur
  1. Crée index.html
    • "Utilise Flexbox pour afficher deux tableaux côte à côte
    • Premier tableau : données après ETL
    • Deuxième tableau : données avec seuil
    • Ajouter style css
